Contributing
============

Welcome, so you are thinking about contributing ?
Awesome, this a great place to start.

Setup
-----

```bash
git clone git@github.com:innovativeinnovation/cheatsheet-latex-template.git
cd cheatsheet-latex-template
```

Build
-----

```bash
make
```
